# itdata_analysis
Dự án phân tích dữ liệu tuyển dụng trong lĩnh vực công nghệ thông tin
- Thời gian: 04/2023 - 08/2023
- Mô tả dự án:
  Xây dựng hệ thống phân tích dữ liệu tuyển dụng trong lĩnh vực công nghệ thông tin bao gồm các công đoạn:
  + Thu thập dữ liệu từ 10 website tuyển dụng có lượng tin tuyển dụng lớn trên internet như CareerBuilder, TopCV, TopDev, ... và lưu trữ dữ liệu thu thập được theo các trường dữ liệu như link công việc, tên công việc, công ty, kinh nghiệm, cấp bậc, lương, yêu cầu công việc, ... dưới định dạng file excel.
  + Đổ dữ liệu của 12 trường quan trọng là ID tin, web, tên công việc, công ty, kinh nghiệm, lương, địa điểm, yêu cầu công việc, ... từ file excel vào cơ sở dữ liệu mysql để lưu trữ dữ liệu và tiến hành ETL dữ liệu. Dữ liệu sẽ được làm sạch, chuẩn hóa và được mô hình hóa thành dữ liệu Dim, Fact của mô hình dữ liệu nhiều chiều OLAP và lưu trữ trong kho dữ liệu.
  + Dữ liệu từ kho dữ liệu sẽ được đưa vào công cụ trực quan hóa dữ liệu PowerBI để xây dựng các báo cáo phân tích tổng quan về tình hình tuyển dụng ngành công nghệ thông tin.
  + Công khai các báo cáo phân tích lên internet để có thể dễ dàng chia sẻ cho mọi người xem và tham khảo.
